# AI-Based Object Detection App for Blind Users

This Android application helps blind users detect nearby objects in real time
using a YOLO TensorFlow Lite model and provides audio feedback using
Text-to-Speech.

## Objectives
- Real-time object detection
- Offline AI processing
- Audio feedback for blind users
- Simple and accessible user interface

## Technologies Used
- Android Studio
- Java
- CameraX API
- TensorFlow Lite
- YOLO
- Text-to-Speech
